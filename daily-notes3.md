# Gün 3

Bugün TaskBoard projesine CSS ekleyerek sayfanın görsel tasarımını geliştirdim. CSS dosyasını HTML dosyasına bağladım ve temel sayfa düzenini oluşturdum. Görev tablosu ile görev ekleme formunu kart görünümüne dönüştürdüm. Öncelik seviyeleri için renkli badge sınıfları ekledim ve butona hover efekti uyguladım. Ayrıca CSS değişkenleri kullanarak tekrar eden renk ve boşluk değerlerini düzenledim. Bu çalışma sayesinde Box Model, temel CSS yapısı ve sayfa düzeni konularında pratik yapmış oldum.
