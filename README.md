# TaskBoard

TaskBoard, kullanıcıların görevlerini oluşturup düzenleyebileceği ve takip edebileceği bir görev yönetim uygulamasıdır.
Proje, uzaktan Full-Stack staj programı kapsamında adım adım geliştirilmektedir.
Uygulamanın kullanıcı arayüzü HTML, CSS ve JavaScript ile geliştirilecektir.
Sunucu tarafı ve iş mantığı ise ASP.NET Core (C#) ile oluşturulacaktır.
Görev ekleme, listeleme, güncelleme ve silme işlemleri uygulamanın temel özelliklerini oluşturacaktır.
Frontend ve backend katmanları ayrı klasörlerde geliştirilerek düzenli bir proje yapısı oluşturulacaktır.
Proje boyunca Git ve GitHub kullanılarak sürüm kontrolü ve düzenli commit yönetimi uygulanacaktır.
Her staj gününde uygulamaya yeni özellikler eklenerek proje aşamalı olarak geliştirilecektir.
Final aşamasında çalışan, düzenli ve sürdürülebilir bir TaskBoard uygulaması ortaya çıkarılması hedeflenmektedir.
